# 개발 블로그
# YoungGil's Tech Blog

로컬 서버
- bundle exec jekyll serve
개인 학습 내용 정리, 기술 지식 공유, 프로젝트 경험 기록을 위한 기술 블로그입니다.
프론트엔드(React, Vue)와 백엔드(PHP, Laravel), 알고리즘, 프로젝트 회고 등을 폭넓게 다룹니다.

## 📚 주요 콘텐츠

- **JavaScript/TypeScript**: 프론트엔드 핵심 기술 및 심화 개념
- **React & Vue**: 프레임워크 동작 원리 및 상태 관리 패턴 (Jotai, Vuex)
- **CSS**: Tailwind CSS, Styled-components 등 스타일링 도구 활용법
- **Backend**: PHP, Laravel 프레임워크 활용 및 서버 아키텍처
- **Algorithm**: 자료구조 및 알고리즘 학습 내용
- **Project**: 프로젝트 진행 과정 및 회고 (KPT 등)

## 🛠 기술 스택

- **Static Site Generator**: Jekyll v4.3.2
- **Hosting**: GitHub Pages
- **Languages**: Markdown, Liquid, HTML, CSS, JavaScript

## 📂 프로젝트 구조

- `_posts/`: 블로그 포스트 마크다운 파일 (카테고리별 분류)
- `_site/`: 빌드된 정적 결과물
- `assets/`: 이미지, CSS, JS 등 정적 리소스

## 🚀 로컬 실행 방법

이 프로젝트는 Jekyll을 사용하여 빌드됩니다.

```bash
# 의존성 설치 및 서버 실행
bundle install
bundle exec jekyll serve
```
